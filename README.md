# CareerCraft

AI Skill Gap & Career Intelligence Platform

## Project Description
CareerCraft is an AI-powered platform that analyzes user skills from resumes, identifies skill gaps, and provides career recommendations and learning roadmaps.

## Tech Stack
- Frontend: React
- Backend: Flask (Python)
- Database: MongoDB
- AI/NLP: spaCy
